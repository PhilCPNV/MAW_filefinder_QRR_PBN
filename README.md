# TurboFinder

**A new file finder made in C#**

## Description 

##### Notre projet est un logiciel de recherche de fichier. Il permet d'effectuer des recherches par :
-	Nom de fichier
-	Extensions
-	Le contenu du fichier
-	Une plage de date
